# Definição de Requisitos do Produto (PRD)

## Descrição do produto

### Problema

* Descrever o problema de negócio que o produto resolve.
* Explicar quem é impactado pelo problema.
* Quantificar impactos quando possível.

**Template:**

> O problema é [DESCREVER PROBLEMA].
>
> Esse problema afeta [PÚBLICO-ALVO], causando [IMPACTOS NEGATIVOS].

### Solução

* Descrever a solução proposta.
* Explicar como o produto resolve o problema.
* Apresentar os principais módulos ou fluxos.

**Template:**

> O [NOME DO PRODUTO] resolve esse problema por meio de [DESCREVER SOLUÇÃO].
>
> Para [TIPO DE USUÁRIO 1]: [DESCREVER BENEFÍCIO].
>
> Para [TIPO DE USUÁRIO 2]: [DESCREVER BENEFÍCIO].

### Diferenciais

* **[DIFERENCIAL 1]**: [DESCRIÇÃO]
